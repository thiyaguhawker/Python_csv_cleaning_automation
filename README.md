# Python_csv_cleaning_automation
This project takes csv file from raw folder (input) and store the cleaned csv in output folder
1. python project file takes csv file from raw input folder and handle null and duplicate values
and save it in ouput file here i have used my local folder for input and output.
2. quality checks folder check whether the data has been cleaned properly by viewing headers
3. batch cleaner file automates the above task
4. we have a bat file to call the batch.py for automation purpose
5. used windows task scheduler to perform the task everyday at 5 am.
